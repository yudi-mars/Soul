Wireless Datasets
========================

The following are wireless signal-related datasets supported by the Soul toolkit:

UT-HAR
-------------------------------------------
Citation：
    https://ieeexplore.ieee.org/document/8067693
Download Link：
    https://github.com/ermongroup/Wifi_Activity_Recognition?tab=readme-ov-file
Detailed Information：

    - CSI data size： 1 x 250 x 90
    - Number of classes： 7
    - Classes： lie down, fall, walk, pickup, run, sit down, stand up
    - Training set size： 3977
    - Test set size： 996
