Acoustic Datasets
========================

The following are the acoustic datasets supported by the Soul toolkit:

GTZAN
-------------------------------------------
Citation:
    https://ieeexplore.ieee.org/abstract/document/1021072
Download Link:
    https://www.kaggle.com/datasets/andradaolteanu/gtzan-dataset-music-genre-classification
Detailed Information:

    - Audio data size： reduce_size x reduce_size
    - Number of classes： 10
    - Classes： blues, classical, country, disco, hiphop, jazz, metal, pop, reggae, rock
    - Training set size： 800
    - Test set size： 200

UrbanSound8K
-------------------------------------------
Citation:
    https://dl.acm.org/doi/10.1145/2647868.2655045
Download Link:
    https://urbansounddataset.weebly.com/download-urbansound8k.html
Detailed Information:

    - Audio data size： reduce_size x reduce_size
    - Number of classes： 10
    - Classes： air_conditioner, children_playing, dog_bark, drilling, engine_idling, jackhammer, siren, car_horn, gun_shot, street_music
    - Training set size： 7079
    - Test set size： 1653

ESC-50
-------------------------------------------
Citation：
    https://dl.acm.org/doi/abs/10.1145/2733373.2806390
Download Link:
    https://github.com/karoldvl/ESC-50/archive/master.zip
Detailed Information:

    - Audio data size： reduce_size x reduce_size
    - Number of classes： 50
    - Classes： Animals , Natural soundscapes & water sounds, Human (non-speech) sounds, Interior/Domestic sounds...
    - Training set size： 1600
    - Test set size： 400

Google Speech Commands
-------------------------------------------
Citation：
    https://arxiv.org/abs/1804.03209
Download Link:
    https://huggingface.co/datasets/google/speech_commands
Detailed Information:

    - Audio data size： reduce_size x reduce_size
    - Number of classes： 35
    - Classes： 
        "yes","no","up","down","left","right","on","off","stop","go","zero","one","two","three","four","five","six","seven","eight","nine",
        "bed","bird","cat","dog","happy","house","marvin","sheila","tree","wow","backward","forward","follow","learn","visual"
    - Training set size： 84663
    - Test set size： 21166
