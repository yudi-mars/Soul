Motion Datasets
====================

The following are motion capture-related datasets supported by the Soul toolkit:

UCI HAR
-------------------------------------------
Citation：
    https://www.sciencedirect.com/science/article/abs/pii/S0925231215010930
Download Link：
    https://archive.ics.uci.edu/dataset/240/human+activity+recognition+using+smartphones
Detailed Information：

    - Motion sensing data size： 9 x 128
    - Number of classes： 6
    - Classes： WALKING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, SITTING, STANDING, LAYING
    - Training set size： 7352
    - Test set size： 2947

HHAR
-------------------------------------------
Citation：
    https://dl.acm.org/doi/10.1145/2809695.2809718
Download Link：
    https://archive.ics.uci.edu/dataset/344/heterogeneity+activity+recognition
Detailed Information：

    - Motion sensing data size： 3 x window_size
    - Number of classes： 6
    - Classes： WALKING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, SITTING, STANDING, LAYING
    - Training set size： 140928
    - Test set size： 35233

MotionSense
-------------------------------------------
Citation：
    https://dl.acm.org/doi/10.1145/3302505.3310068
Download Link：
    https://www.kaggle.com/datasets/malekzadeh/motionsense-dataset
Detailed Information：

    - Motion sensing data size： 12 x window_size
    - Number of classes： 6
    - Classes： Downstairs, Upstairs, Walking, Jogging, Sitting, Standing
    - Training set size： 17491
    - Test set size： 4373

Shoaib
-------------------------------------------
Citation：
    https://www.mdpi.com/1424-8220/14/6/10146
Download Link：
    https://www.researchgate.net/publication/266384007_Sensors_Activity_Recognition_DataSet
Detailed Information：

    - Motion sensing data size： 12 x window_size
    - Number of classes： 8
    - Classes： Walking, Standing, Jogging, Sitting, Biking, Walking upstairs, Walking downstairs, Invalid data
    - Training set size： 10451
    - Test set size： 4479
